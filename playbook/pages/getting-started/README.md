# Getting Started

This section explains the foundational concepts and workflow for modernising legacy systems using AI technologies. It covers the four pillars of legacy modernisation, the workflow to follow, and sets the stage for the pathways that provide specific guidance for different technologies.

**Important:** Only use AI tools from the [Defra Approved Tools](../appendix/defra-approved-tools) list.

**Important:** Enable privacy settings to protect government data. Privacy settings stop your code and data being stored on AI providers’ servers. They also prevent your data being used to train AI models.
